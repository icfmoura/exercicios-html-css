Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir.
Passe o mause sobre os termos para descobrir qual tag deverá utilizar em cada caso

  • Parágrafos
  • Titulos
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas numeradas
  • Listas com demarcadores

Veja a seguir o código para criar uma lista

            <ol type="1">
                <li>item 1</li>
                <li>item 2</li>
                <li>item 3</li>
            <ol/>
